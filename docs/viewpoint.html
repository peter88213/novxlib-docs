The <Viewpoint> tag

<Viewpoint>

Purpose

Defines a section’s viewpoint character.

Attributes

  • id

The section’s viewpoint can be entered in the section’s properties view, or via menu entries in novelibre. The viewpoint is optional.

Example:

<Viewpoint id="cr2" />

The id attribute

This is a reference to an existing character. This attribute is required. The character ID consists of the character prefix cr and a number.
