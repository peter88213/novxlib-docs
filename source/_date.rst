==============
The <date> tag
==============

.. admonition:: <date>
   
   Purpose
      Defines creation date of a comment.

Date format: *YYYY-MM-DDThh:mm:ss*, according to ISO 8601.

Example:

.. code-block:: xml

   <date>2024-05-30T13:04:04.34</date>
   
.. hint:: 

   This is usually generated by the *Writer* word processor.
