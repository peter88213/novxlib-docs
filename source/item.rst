==============
The <ITEM> tag
==============

.. admonition:: <ITEM>
   
   Purpose
      Defines an item in the story world.

   Attributes
      - `id <#the-id-attribute>`__

   Content
      - `Title <title.html>`__
      - `Desc <desc.html>`__
      - `Link <link.html>`__
      - `Notes <notes.html>`__
      - `Tags <tags.html>`__
      - `Aka <aka.html>`__

The id attribute
----------------

This attribute is required. The item ID consists of the
item prefix **it** and a number.

Example: ``it13``
