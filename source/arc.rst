=============
The <ARC> tag
=============

.. admonition:: <ARC>
   
   Purpose
      Defines a plot line.

   Attributes
      - `id <#the-id-attribute>`__

   Content
      - `Title <title.html>`__
      - `Desc <desc.html>`__
      - `Link <link.html>`__
      - `Notes <notes.html>`__
      - `ShortName <shortname.html>`__
      - `Sections <sections.html>`__
      - `POINT <point.html>`__

The id attribute
----------------

This attribute is required. The plot line ID consists of the
plot line prefix **ac** and a number.

Example: ``ac13``
