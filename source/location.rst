==================
The <LOCATION> tag
==================

.. admonition:: <LOCATION>
   
   Purpose
      Defines a location in the story world.

   Attributes
      - `id <#the-id-attribute>`__

   Content
      - `Title <title.html>`__
      - `Desc <desc.html>`__
      - `Link <link.html>`__
      - `Notes <notes.html>`__
      - `Tags <tags.html>`__
      - `Aka <aka.html>`__

The id attribute
----------------

This attribute is required. The location ID consists of the
location prefix **lc** and a number.

Example: ``lc13``
