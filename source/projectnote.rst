=====================
The <PROJECTNOTE> tag
=====================

.. admonition:: <PROJECTNOTE>
   
   Purpose
      Defines a project note.

   Attributes
      - `id <#the-id-attribute>`__

   Content
      - `Title <title.html>`__
      - `Desc <desc.html>`__
      - `Link <link.html>`__

The id attribute
----------------

This attribute is required. The project note ID consists of the
project note prefix **pn** and a number.

Example: ``pn13``
