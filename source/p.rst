===========
The <p> tag
===========

.. admonition:: <p>
   
   Purpose
      Defines a paragraph.

   Attributes
      - `style <#the-style-attribute>`__
      - `lang <#the-lang-attribute>`__

   Content
      - `em <em.html>`__
      - `strong <strong.html>`__
      - `span <span.html>`__
      - `comment <comment.html>`__
      - `note <note.html>`__
      - Plain text

All multiline text is put into paragraphs.

The style attribute
-------------------

- quotations:

Default value: None

The lang attribute
------------------

Default: implied
