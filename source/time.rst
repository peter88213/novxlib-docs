==============
The <Time> tag
==============

.. admonition:: <Time>
   
   Purpose
      Defines the section's narrative start time.

   Content
      Plain text 

The time information is optional.

Time format: *hh:mm:ss*, according to ISO 8601.

.. hint::

   The seconds are not displayed in *novelibre*. 

Example:

.. code-block:: xml

   <Time>01:05:00</Time>
