==============
The <Time> tag
==============

.. admonition:: <Time>
   
   Purpose
      Defines a time.

   Content
      - Plain Text 


Time format: *hh:mm:ss*, according to ISO 8601.

.. hint::

   The seconds are not displayed in *novelibre*. 

Example:

.. code-block:: xml

   <Time>01:05:00</Time>
